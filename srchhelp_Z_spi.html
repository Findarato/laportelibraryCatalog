Introduzca en Palabras para Buscar: EJEMPLOS
ADYACENCIA Para buscar múltiples palabras juntas como una frase, encierrelas entre comillas. Las búsquedas por defecto devolverán registros que contengan cualquiera de los términos de búsqueda introducidos (OR, por defecto).
  • "calentamiento global" (encuentra sólo registros que contienen las palabras "calentamiento global" como una frase)
  • polucion cancer (encuentra cualquier registro que contiene "polución" y cualquier registro que contiene "cancer")
TRUNCAMIENTO Las Palabras pueden ser truncadas por la derecha usando un asterisco. Use un asterisco simple para truncar entre 1-5 caracteres. Use doble asterisco para un truncamiento completo.
  • pobla*
  • dost** (encontraría, entre otros, "Dostoyevsky")
OPERADORES Estos no funcionarán igual que los operadores Booleanos; sólo afectarán a una palabra y no modificarán las relaciones entre las palabras. Use un + para requerir que una palabra esté presente, use un - para excluir una palabra, si no se especifica el operador que se usa es OR.
  • nuclear +familia (encuentra cualquier registro con "nuclear" Y "familia", y cualquier registro con "familia")
  • +nuclear +familia (encuentra cualquier registro con "nuclear" Y "familia")
  • nuclear -familia (encuentra cualquier registro con "nuclear" Y NO "familia" y cualquier registro sin "familia")
  • +nuclear -familia (encuentra cualquier registro con "nuclear" Y NO "familia")
